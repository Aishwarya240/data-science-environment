# Data Science Environment

This repository demonstrates my Python Data Science environment setup.

## Environment

- Conda environment: myenv
- Python 3.12
- Packages: numpy, pandas, matplotlib, seaborn, scikit-learn, plotly, jupyter

## Usage

Activate environment and run Jupyter:

```bash
bash -c "source ~/miniconda3/etc/profile.d/conda.sh && conda activate myenv && jupyter notebook"
```

Select the kernel **Python (myenv)** in Jupyter to run notebooks with the correct packages.
